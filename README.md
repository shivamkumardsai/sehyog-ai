# 🩺 SEHYOG AI

SEHYOG AI is an AI-powered healthcare assistant that helps users understand their medical reports through AI-driven analysis. Users can upload PDF medical reports, receive simplified health insights, interact with an AI chatbot, and monitor report history via a dashboard.

## ✨ Features

- 📄 Upload Medical Reports (PDF)
- 🤖 AI Report Analysis using Gemini AI
- 💬 AI Healthcare Chatbot
- 📊 Dashboard with Analytics
- 🗂 Report History
- ☁ MongoDB Atlas Database
- ⚡ Fast Next.js Application

## ⚙ Environment Variables

Create a `.env.local` file:

```env
DATABASE_URL=your_mongodb_connection_string

GEMINI_API_KEY=your_gemini_api_key
```

---

## Installation

```bash
git clone https://github.com/shivamkumardsai/sehyog-ai.git

cd sehyog-ai

npm install

npx prisma generate

npx prisma db push

npm run dev
```
